# Broken Music Player

一个浏览器本地播放器

主要特性：

0. 仅支持 Chrome > 51
1. 支持离线使用
2. 缓存已添加的音乐文件到浏览器

立即使用：

- 国内：https://dgeibi.gitee.io/static/player/
- 全球： https://player.dgeibi.xyz/

## 构建

安装 [Node.js](https://nodejs.org/)

安装依赖：`npm install`

构建生产环境版本：`npm run build`，结果在`dist`

构建开发环境版本并启动服务器：`npm run start`

构建生产环境版本并启动服务器：`npm run start:prod`
